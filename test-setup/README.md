# ESPORTS STEAM TOOLS - TEST ENVIRONMENT

This folder contains a complete test setup for the Esports Steam Tools.

## Quick Start

1. **Copy the entire `test-setup` folder to your Windows machine**
2. **Run `TEST-SETUP.bat` as Administrator**
3. **Follow the on-screen instructions**

## Test Files Included

- `TEST-SETUP.bat` - Main setup script for Windows
- `test-config.env` - Sample configuration with test credentials  
- `test-steam-tools.bat` - Modified version for testing
- `TEST-INSTRUCTIONS.txt` - Detailed testing guide

## What Gets Tested

✅ Windows compatibility  
✅ Configuration loading  
✅ Steam path detection  
✅ Account selection  
✅ Game selection  
✅ Console stays open for debugging  
✅ All error handling  

## Security Notes

- Test credentials are fake and safe to use
- No real Steam accounts will be affected
- Steam will not actually login with test credentials
- Perfect for testing the interface and logic

## Support

If you encounter issues:
1. Run as Administrator
2. Check Windows version compatibility
3. Ensure Steam is installed (for path detection)
4. Review the console output for errors
